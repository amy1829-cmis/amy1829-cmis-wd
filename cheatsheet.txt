Terminal Commands-
ls <directory name>:  List contents of a directory
cd: Go home.
cd <directory name>:  Change directory
cd ..: Go up
cat <file name>: View the contents of a file
gedit <file name> &
python -m SimpleHTTPServer - Start the development serve
rmdir <file name>: Remove a file in repository
mkdir <file name>: Create a new file in repository
cd ..: Use when you want to go back to the file before
Ctrl c: Use when you want the green/blue thing back
